Tamaño de fuentes en función del viewport

La ventana gráfica (o viewport) es el área visible de tu página en el navegador que utilizas para ver un sitio, que tiene un tamaño determinado. En CSS hay unidades que asociadas con el tamaño de la ventana gráfica: las unidades vw para el ancho y vh para la altura. Con estas unidades podemos establecer para el texto (u otros elementos, como veremos más adelante) tamaños relativos a la ventana gráfica del usuario.
